Guide to Install Ubuntu 24.04 LTS on MacBook Pro & Air

Pre-Installation Checklist

  • Backup your data (Time Machine or external drive)
  • 16 GB+ USB flash drive (for the Ubuntu installer)
  • Ethernet
  • USB-C hub (for newer Macs without USB-A)
  • Keyboard & mouse (if you're wiping macOS completely)

For Intel-based MacBook Pro / Air

1. Download Ubuntu 24.04 LTS

Go to https://www.releases.ubuntu.com/noble/, choose Ubuntu Desktop 24.04 LTS (64-bit) and download the ISO.

2. Create a Bootable USB

Use balenaEtcher (Mac/Windows/Linux):

  • Open Etcher → Select the Ubuntu ISO → Select your USB drive → Click "Flash!"

3. Disable Secure Boot (if applicable)

Restart and hold Command + R to enter macOS Recovery.

  • Go to Utilities → Startup Security Utility
  • Select "Allow booting from external or removable media."

4. Boot from USB

Insert the USB, restart your Mac, and hold Option (⌥) at startup. Select the USB labeled "EFI Boot".

5. Install Ubuntu

  • Choose “Try Ubuntu” or go directly to “Install Ubuntu.”
  • Partition your drive or erase macOS if you want Ubuntu as the only OS.
  • Follow on-screen instructions to finish the installation.

6. Fix Boot Graphics Issues (Optional but Recommended)

If Ubuntu boots to a black screen or you're seeing glitches after installation, update GRUB to add nomodeset, which helps with compatibility:

  1. Boot into Ubuntu
  2. Open Terminal and run:
    3. sudo gedit /etc/default/grub
  3. Find the line: 
    GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
  4. Change it to: 
    GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nomodeset"
  5. Save and close the file.
  6. Update GRUB with: 
    sudo update-grub
  7. Reboot: 
    sudo reboot

Why? The nomodeset option tells Ubuntu to avoid using certain graphics drivers during boot, which helps resolve black screen or boot hangs on some MacBook GPUs.
